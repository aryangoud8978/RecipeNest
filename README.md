# 🍔 RecipeNest

RecipeNest is a modern Android Recipe App built using Jetpack Compose in Android Studio.

---

# ✨ Features

- Splash Screen
- Modern UI Design
- Search Bar
- Recipe Cards
- Clickable Recipes
- Scrollable Layout
- Clean User Interface
